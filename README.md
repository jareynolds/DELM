# DELM
design experience language model for creating any UI element or component

ESLM - UI Design Small Language Model
A React application for managing and interacting with an Enterprise Small Language Model for UI Design generation.

Features
Dashboard: Overview of capabilities, enablers, stories, and service health
Capabilities Management: Track and manage system capabilities with dependencies
SLM Engine: Generate UI components from natural language prompts
Prompts: History and management of user prompts
Health Monitoring: Real-time service health status
Stories: User story management and progress tracking
AI Governance: Preset 2 (Guided Recommendations) governance controls
Settings: SLM configuration and system settings
Tech Stack
React 18.2
TypeScript
Vite
Tailwind CSS
React Router DOM
Axios
Lucide React Icons
Getting Started
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
Project Structure
